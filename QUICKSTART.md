# ⚡ Quick Start Guide

## Get Running in 2 Minutes

### Option 1: GitHub Pages (Recommended)

1. **Upload to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR-USERNAME/btq-companion-v8.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings → Pages
   - Select `gh-pages` branch
   - Save

3. **Done!** 
   - Visit: `https://YOUR-USERNAME.github.io/btq-companion-v8/`

### Option 2: Run Locally

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## First Game

1. **Ships Tab**: Create 2 ships
   - Click "Add Ship"
   - Set nationality, crew, guns
   - Click "Add to Roster"

2. **Movement Tab**: Calculate speeds
   - Check current wind
   - View each ship's speed

3. **Gunnery Tab**: Fire!
   - Select firing ship
   - Select target
   - Choose range & shot type
   - Roll!

4. **Damage Tab**: Apply results
   - Enter hull damage
   - Record crew losses
   - Track fires

5. **Advance Turn**: Click "Advance Turn"
   - Wind may shift
   - Fires spread
   - Turn counter increases
